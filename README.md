# AI Resume - 智能简历助手小程序

## 项目简介
这是一个基于微信小程序云开发的 AI 简历助手应用，帮助用户快速生成、优化和导出专业简历。

## 技术栈
- 微信小程序
- 微信云开发

## 项目结构
```
ai-resume/
├── pages/          # 页面目录
│   └── index/      # 首页
├── cloudfunctions/ # 云函数目录
│   └── login/      # 登录云函数
├── utils/          # 工具函数
├── images/         # 图片资源
├── app.js          # 小程序主入口
├── app.json        # 小程序配置
├── app.wxss        # 全局样式
├── project.config.json  # 项目配置
└── sitemap.json    # 站点地图配置
```

## 开发步骤

### 1. 打开项目
使用微信开发者工具打开 ai-resume 目录

### 2. 开通云开发
- 点击工具栏的"云开发"按钮
- 按照提示开通云开发服务

### 3. 创建云环境
- 在云开发控制台创建环境

### 4. 上传云函数
- 在 cloudfunctions/login 目录右键
- 选择"上传并部署：云端安装依赖"

### 5. 运行项目
- 点击编译按钮运行项目

## 功能规划
- 简历生成
- 简历优化
- 简历导出
